Sobre o que se trata?

Jokenpow é um jogo de luta, sabe aquele momento que você precisa resolver um dilema de maneira imparcial e resolve decidir no Pedra, Papel, Tesoura? Aqui funciona do mesmo jeito, mas, não dependemos da sorte, vence aquele que melhor usa a sua escolha.

Teclas Importantes

Player 1

Na seleção de personagens, use W e S para escolher seu personagem.

O player 1 se movimenta utilizando as teclas:

(W) para pular

(A) para andar para esquerda

(D) para andar para direita

(C) para atacar

Player 2

Na seleção de personagens, use I e K para escolher seu personagem.

O player 2 se movimenta utilizando as teclas:

(I) para pular

(J) para andar para esquerda

(L) para andar para direita

(N) para atacar

Boa luta!
